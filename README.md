# dotfiles

## Premise

`git` and `vim` is installed.

## Usage

```
$ cd ~
$ git clone https://github.com/taji-taji/dotfiles.git
$ sh dotfiles/install.sh
```

#### Install languages

```
$ sh dotfiles/tools/pyenv.sh
$ sh dotfiles/tools/rbenv.sh
$ sh dotfiles/tools/swiftenv.sh
```

## Environment Variables

- `$GOPATH: $HOME/go`
